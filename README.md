# Systemy operacyjne 2
Repozytorium do przechowywania zadań rozwiązywanaych w ramach przedmiotu systemy operacyjne 2 (2020/2021, semestr V).

## Laboratoria zawierają:

### Lab_1: 
Program InputOutput (odczytanie i wypisanie wartości double, odczytanie i wypisanie napisu przy pomocy funkcji scanf oraz fgets),
program obliczający ciąg fibonacciego kilkoma sposobami oraz program FiboTree wypisujący drzewo wywołań ciągu fibonacciego.

### Lab_2:
Program Sequence działający analogicznie jak programy dla ciągu fibonnaciego oraz program SequenceTree.

### Lab_3:
Program Fractions (działanie na strukturze reprezentującej ułamki), funkcja indexOf, funkcja identity (zwracająca imię i nazwisko oddzielone spacją), funkcja login tworząca login użytkownika na podstawie pierwszej litery imienia i całego nazwiska.

### Lab_4:
Pełna implementację programu Fractions, funkcja isNumber (sprawdzającą, czy napis s przechowuje liczbę całkowitą), funkcja trim (usuwającą z napisu s początkowe i końcowe znaki białe), funkcja getOperator (określająca operator z napisu s).

### Lab_5:
Program Files do odczytu zawartości plików, program CountWords do zliczania słów, program DisplayWords (odczytujący plik tekstowy oraz wypisujący poszczególne słowa), program NewLine (testujący jakie znaki określają koniec linii w pliku tekstowym na systemie Windows i Linux).

### Lab_6:
Program dodający oraz program usuwający użytkowników w systemie linux, programy testujące funkcje crypt do szyfrowania haseł.

### Lab_7:
Program check (sprawdzający na jakim systemie został skompilowany), program arguments (wypisujący kolejno: wartość zmiennej argc, adres programu, nazwę programu, przekazane parametry), program list (wywołujący komendę dir lub ls), program cyfry (wyliczający liczbę cyfr danej liczby), program realizujący dodawanie pisemne.

### Lab_8:
Programy do podstawowej obsługi procesów.

### Lab_9:
Program regexp (ilustrujący wyrażenia regularne POSIX’a), funkcja losuj (zwraca wartość losową z przedziału), funkcje wariancje, funkcja set (w sposób losowy określa wartość parametru salt dla funkcji crypt_r), errnoExample i perrorExample (wypisanie błędu otwarcia pliku przy pomocy zmiennej globalnej errno i perror).

### Lab_10:
Programy do operowania na plikach za pomocą parametrów. Folder zawiera min. zadania z: kopiowanie zawartości pliku, zmieniający kolejność znaków w każdej linii pliku,
program do rozpoznawania czy plik o podanej nazwie jest plikiem tekstowym, łączenia plików, zliczający liczbę słów, porównywania plików.
